Editor Tools

1 - TrashMobsTools

A custom Unreal Engine editor extension developed for a VR MMORPG studio in Los Angeles. This tool allows designers and developers to set up loot drops, analyze item distribution, and debug player inventory directly inside Unreal Engine.

  • Loot & Drop Management: Define item drops, probabilities, and boss rewards with a visual interface.
  • Data Integration: Pulls backend server data into Unreal for real-time analysis.
  • Item & Avatar Tools: Manage dropped items, avatars, and avatar parts with filtering and probability settings.
  • Photo Studio: Generate high-quality item thumbnails/icons for consistent in-game UI.
  • Account Analyzer: Debug player accounts by reviewing loot histories and drop reports.

This streamlined workflow eliminated the need for developers to switch between Unreal and external dashboards, significantly speeding up iteration and debugging.

2 - Riff XR Avatar SDK

An Unreal Engine editor toolkit designed to streamline the user-generated content (UGC) pipeline for the VR rhythm game Riff XR. This SDK Assistant guides modders through a multi-step wizard, automating the complex setup of IK Rigs and Retargeters to make custom avatars game-ready.

  • Guided Setup Wizard: A step-by-step interface ("SDK Assistant") that walks creators through the entire process, from mesh preparation and bone mapping to final asset generation.
  • Automated Rig Generation: Analyzes the avatar's skeleton and automatically creates the required IKRig and IKRetargeter assets, a process that previously had to be done manually.
  • Intuitive Bone Mapping: Provides a clear UI for mapping the custom avatar's skeleton to the Riff XR standard, with warnings and hints to help correct common setup issues.
  • Custom VR Animation Nodes: Includes a dedicated suite of Animation Blueprint nodes like "RiffXR Hands Pose Generator" and "RiffXR Twist Control" to simplify hand posing and solve common arm twisting issues in VR.

By automating the technical rigging process, the SDK significantly lowers the barrier to entry for community creators, allowing them to focus on avatar design rather than complex engine setups.
