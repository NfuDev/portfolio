Editor Tools

1 - TrashMobsTools

A custom Unreal Engine editor extension developed for a VR MMORPG studio in Los Angeles. This tool allows designers and developers to set up loot drops, analyze item distribution, and debug player inventory directly inside Unreal Engine.

  • Loot & Drop Management: Define item drops, probabilities, and boss rewards with a visual interface.
  • Data Integration: Pulls backend server data into Unreal for real-time analysis.
  • Item & Avatar Tools: Manage dropped items, avatars, and avatar parts with filtering and probability settings.
  • Photo Studio: Generate high-quality item thumbnails/icons for consistent in-game UI.
  • Account Analyzer: Debug player accounts by reviewing loot histories and drop reports.

This streamlined workflow eliminated the need for developers to switch between Unreal and external dashboards, significantly speeding up iteration and debugging.

2 - Riff XR Avatar SDK

An Unreal Engine editor toolkit designed to streamline the user-generated content (UGC) pipeline for the VR rhythm game Riff XR. This SDK Assistant guides modders through a multi-step wizard, automating the complex setup of IK Rigs and Retargeters to make custom avatars game-ready.

  • Guided Setup Wizard: A step-by-step interface ("SDK Assistant") that walks creators through the entire process, from mesh preparation and bone mapping to final asset generation.
  • Automated Rig Generation: Analyzes the avatar's skeleton and automatically creates the required IKRig and IKRetargeter assets, a process that previously had to be done manually.
  • Intuitive Bone Mapping: Provides a clear UI for mapping the custom avatar's skeleton to the Riff XR standard, with warnings and hints to help correct common setup issues.
  • Custom VR Animation Nodes: Includes a dedicated suite of Animation Blueprint nodes like "RiffXR Hands Pose Generator" and "RiffXR Twist Control" to simplify hand posing and solve common arm twisting issues in VR.

By automating the technical rigging process, the SDK significantly lowers the barrier to entry for community creators, allowing them to focus on avatar design rather than complex engine setups.

3 - Unreal Engine Plugins Builder

A standalone C++ desktop application built with the wxWidgets framework to streamline the packaging and deployment of Unreal Engine plugins. Epic Games requires all marketplace plugins to be built via the command-line Build Automation Tool (UAT), so I created this application to provide a simple, powerful graphical interface for that process.

  • Centralized Plugin Management: Add, track, and manage all your Unreal Engine plugins in one place, complete with version numbers and build history.
  • One-Click UAT Builds: Package plugins with a single click, directly invoking Unreal's Build Automation Tool without needing to use the command line.
  • Multi-Version Engine Targeting: Easily select and build your plugins against different versions of Unreal Engine to ensure broad compatibility.
  • Integrated Build Console: Monitor the entire packaging process in real-time with a built-in console that displays live output from UAT for quick debugging.
  • Custom Build Arguments: Pass custom command-line arguments to the build process for advanced packaging requirements.

This tool eliminates the tedious and error-prone process of manual command-line builds, ensuring every plugin is packaged correctly for the marketplace and streamlining the release workflow.

Game Dev Tools

1 - Goap AI System

My implementation of a Goal-Oriented Action Planning (GOAP) system for Unreal Engine provides a powerful and flexible framework for creating intelligent AI. This plugin simplifies complex AI behaviors by allowing developers to define a set of simple actions, and the GOAP brain component automatically chains them together to achieve a specific goal. This system also includes a robust editor tool for debugging and visualizing the AI's decision-making process.

Key Features

  • Modular AI Behavior: Instead of writing complex state machines, developers can create minimal, reusable "actions" (e.g., go to a location, play an animation). The GOAP brain component then uses a planner to intelligently chain these actions into a complete plan to achieve a desired goal.
  • Intuitive Setup with Gameplay Tags: The system uses Unreal's native Gameplay Tags to define goals, world states, and agent facts, ensuring a user-friendly setup and preventing spelling errors. By simply adding more actions to the available array, the AI's behavior becomes more sophisticated without any additional programming.
  • Dynamic Planning and Goal Prioritization: The GOAP brain dynamically creates a plan based on the current world state and the agent's facts. The system can also be configured to prioritize certain goals over others (e.g., harvesting mature crops before planting new ones), ensuring the AI's behavior is always relevant and efficient.
  • Built-in Debugging Tools: The plugin includes a comprehensive "Brain Inspector" editor tool that allows developers to visualize the AI's thought process. This tool displays the current plan, preconditions, and effects of each action, as well as a history of plans and tool tip over each action node giving failure results that are also customizable in the action class, making it easy to understand and troubleshoot AI behavior.

This GOAP tool offers an efficient and scalable solution for building complex AI, allowing developers to focus on creating rich and believable agent behaviors with a minimal amount of effort.

2 - Time Stone Plugin

A powerful gameplay plugin for Unreal Engine that gives developers a component-based system to manipulate time. The Time Stone Plugin allows for granular control over the flow of time, enabling mechanics like slowing, stopping, and rewinding the game world on a global or per-actor basis.

  • Comprehensive Time Control: Easily implement slow-motion, pause, and rewind functionalities through a single, versatile component.
  • Actor Isolation System: Isolate specific characters or objects from the global timeline, allowing them to move at normal speed while the world around them is altered.
  • "Ghost" Rewind System: The plugin's standout feature allows actors to spawn a "ghost" of their past selves after a rewind. This ghost replays the exact actions that were just rewound, creating a duplicate actor on the timeline.

3- Arrows Missions System

A flexible and powerful mission system for Unreal Engine, the Arrows Missions System (AMS) provides an easy-to-use framework for creating complex in-game missions. It features a component-based design with dedicated classes for missions and actions, allowing developers to implement custom logic and streamline their workflow. The plugin's standout feature is its ability to save and load game progress, including the state of individual actors and mission objectives, ensuring a seamless experience for players.

  • Blueprintable Mission & Action Classes: The system includes blueprintable mission and action classes with predefined events (e.g., Mission Start, Mission End, Mission Tick) that users can easily hook into for custom logic.
  • Mission Tracking: A centralized mission tracker allows developers to monitor and manage all active missions in one place.
  • Persistent Game State: The system utilizes a payload to save and load mission progress, remembering the state of enemies and assets within the game, even after a player exits and reloads.
  • Dynamic Mission Control: The AMS can dynamically manage mission-related assets, automatically removing them from the game world once a mission is complete to prevent clutter.
  • Location-Based Missions: Easily create missions that are triggered or progressed based on the player's location.

This tool simplifies the process of creating engaging and complex missions, saving countless hours of complex programming and ensuring a robust, customizable system for a variety of game genres.
